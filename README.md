# 概要
Reactのチュートリアルです
https://reactjs.org/tutorial/tutorial.html

# 手順

+ vue-cli のインストール

```
npm install -g vue-cli
```

+ プロジェクトの作成

```
vue init webpack my-project
```

インストール完了後

```
cd my-project
npm run dev
```

ブラウザで http://localhost:8080/
